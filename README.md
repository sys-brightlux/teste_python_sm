# **Teste Rápido de Programação \- Desenvolvimento de Sistemas**

Bem-vindo(a) ao nosso teste\! O objetivo é avaliar sua lógica de programação e familiaridade com as tecnologias que usamos no dia a dia.

Este teste é dividido em duas partes:

1. **Lógica em Python** (arquivo: teste\_python.py)  
2. **Front-End em HTML/JavaScript** (arquivos: index.html e script.js)

## **Contexto**

Estamos trabalhando em um sistema de gerenciamento de projetos. A base do sistema são Usuarios e suas Tarefas. Os dados de exemplo já estão fornecidos nos arquivos de script.

## **Tarefa 1: Lógica de Programação (Python)**

**Arquivo:** teste\_python.py

Abra o arquivo teste\_python.py. Você encontrará uma lista de tarefas e uma função calcular\_progresso(tarefas).

Seu objetivo:  
Complete a função calcular\_progresso para que ela receba a lista de tarefas e retorne a quantidade de tarefas e porcentagem (um float) de tarefas que estão com o status 'concluida'.

* O arquivo já possui os dados de exemplo.  
* Para os dados de exemplo, o resultado esperado é  5 tarefas e 40.0%.

## **Tarefa 2: Front-End (HTML/JavaScript)**

**Arquivos:** index.html e script.js

Abra o index.html no seu navegador. Ele está, por enquanto, vazio.

Seu objetivo:  
Edite o arquivo script.js para que ele popule a lista de tarefas no HTML.

1. Os dados de exemplo (listaTarefas) já estão no script.js.  
2. Você deve usar JavaScript para iterar sobre essa lista.  
3. Para cada tarefa, crie um item de lista (\<li\>) e o adicione na \<ul\> que tem o ID lista-tarefas.  
4. Adicione ao \<li\> a classe CSS correspondente ao status (concluida ou pendente). (O index.html já tem estilos básicos para essas classes).

## **Como Entregar**

Complete as tarefas nos arquivos indicados. Após finalizar, nos avise para que possamos revisar. O tempo para realização do teste é de 1 hora.

Boa sorte\!
